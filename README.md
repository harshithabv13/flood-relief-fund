# Emergency Relief Crowdfunding Platform

An open-source frontend web application built to facilitate and track donations for communities affected by natural disaster situations.

## 🚀 Key Features
* **Informational Hub:** An interactive landing page highlighting flood incident conditions to drive user engagement.
* **Streamlined Donation Form:** A simple, interactive interface to gather donor metadata and simulate transaction checkpoints.
* **Live Contribution Tracker:** Utilizes browser local storage to calculate total cumulative donation metrics and dynamically display a running log of recent backers on confirmation.

## 📁 Project Structure
The project contains the following operational assets:
* `index.html` - The central landing page showcasing crisis relief context and operational gallery view.
* `login.html` - Secure contribution input portal equipped with UPI scan components.
* `confirm.html` - The real-time transaction summary script interface.
* `style.css` - Custom styling declarations handling global responsive layouts.

## 🛠️ How to Launch Locally
1. Download or clone this repository to your local machine.
2. Ensure all asset references remain intact inside the directory.
3. Double-click or open `index.html` in any modern web browser to run the platform locally.
